# Codex Project Memory

## 项目概述
CLI 代码 agent，类 OpenAI Codex，支持任何 OpenAI 兼容的国产 API。

## 技术栈
- **语言**: Go 1.21+
- **CLI 框架**: Cobra + Viper
- **交互**: chzyer/readline
- **协议**: OpenAI Chat Completions（流式 SSE）

## 目录结构
```
agent/   - agent 循环、工具集、undo、session、compression、memory
llm/     - OpenAI 兼容 HTTP 客户端
config/  - 配置管理 (~/.codex/config.yaml)
cmd/     - Cobra CLI 入口
```

## 工具集（10 个）
read_file, write_file, patch_file, list_files, shell_exec, grep_files,
git_status, git_diff, git_log, git_commit

## 编码规范
- 错误统一用 `fmt.Errorf("context: %w", err)` 包装
- tool 实现返回 `ToolResult{Content, IsError}`，不 panic
- 危险操作（shell_exec、git_commit、patch_file）走 approver 审批
- 新工具：① 加 def*() ② 加 Execute switch ③ 加到 NewToolRegistry defs 列表

## 禁忌
- 不引入外部 HTTP 框架（用标准库 net/http）
- 不在工具层直接 os.Exit
- git_commit 必须走 approver，不能静默提交
